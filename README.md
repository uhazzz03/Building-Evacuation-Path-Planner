# Building-Evacuation-Path-Planner
An ASP.NET project using C++ and algorithms related to finding the optimal solution to the shortest path accordingly.

## Folder structure (Present)

evacuatex/  
├─ .gitignore  
├─ README.md  
├─ .vscode/  
│  ├─ launch.json                # VS Code debug config (adds native DLL path)  
│  └─ tasks.json                 # VS Code build tasks (native + API)  
└─ backend/  
   ├─ NativeInterop/  
   │  └─ EvacuateX.Native/  
   │     ├─ CMakeLists.txt       # CMake project for native library  
   │     ├─ evacuatex_native.h   # (C++) header  
   │     ├─ evacuatex_native.cpp # (C++) implementation  
   │     └─ build/               # (generated) build outputs (e.g., Release/Debug)  
   └─ EvacuateX.Api/  
      ├─ Program.cs              # (C#) ASP.NET  Core minimal API  
      ├─ EvacuateX.Api.csproj  
      ├─ Properties/  
      │  └─ launchSettings.json  # (auto-generated by dotnet new)  
      └─ bin/                    # (generated) build outputs  
         └─ Debug/ or Release/
