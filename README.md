# Building-Evacuation-Path-Planner
An ASP.NET project using C++ and algorithms related to finding the optimal solution to the shortest path accordingly.

## Description
- **C++17**: Native pathfinding engine (A*, grid-based).
- **ASP.NET Core 8**: Web API layer with DTO validation, P/Invoke to C++.
- **React/Next.js**: Interactive UI to design buildings, run evacuation simulations, and visualize results.
- **SQL Database**: Persistence for building/floor layouts and simulation results.
- **Docker + CI/CD**: For deployment and reproducible builds.

## Prerequisites
- [CMake](https://cmake.org/download/) ≥ 3.22
- [Visual Studio Build Tools 2022](https://visualstudio.microsoft.com/downloads/) (Windows) or GCC/Clang (Linux/macOS)
- [.NET SDK 8](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
- [Node.js](https://nodejs.org/) ≥ 18 (for frontend, later)
